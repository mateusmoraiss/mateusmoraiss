## Discover my Projects! 🚀 (Eng)

Welcome to my GitHub profile! Explore a collection of my projects, experiments, and learning resources. One project that holds a special place in my coding journey is [Projeto Pizzaria](https://github.com/mateusmoraiss/projeto-pizzaria-C). It marked the beginning of my passion for technology.


I believe that technology will be fundamental for creation in the future. Since college, I have started using ChatGPT for various tasks. Although my projects demonstrate my own knowledge, I admit to using tools as part of the process. I disagree with the academic view that disapproves of this use, as I believe it should focus on being efficient, creative, and functional. If automation ever becomes complete, why not take advantage of it? Some of my projects are 100% programmed by me, especially those before 2019, for instance the [PP Pizzas](https://github.com/mateusmoraiss/projeto-pizzaria-C), when I didn't have access to ChatGPT. That's why in each project on my GitHub, there will be a folder called 'TheFuture,' where I share if I used Artificial Intelligence, made Templates, and give their due credits.
The content will basically be:

